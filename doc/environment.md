###yarn

安装
```
npm -g install yarn@1.3.2
```

命令参照
```
npm install => yarn
npm init => yarn
npm install xxx -g => yarn global add xxx
npm install xxx --save => yarn add xxx
npm install xxx --save-dev => yarn add xxx --dev
npm uninstall xxx => yarn remove xxx
npm run xxx => yarn run xxx
```
